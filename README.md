# Banco Digital - API RESTful
Desafio Módulo 2 - Cubos Academy - Back-end

## Descrição:
Trata-se de um projeto de um banco digital com cadastro, alteração e exclusão de usuários, além de transações financeiras básicas como saque, depósito, transferências, além  apresentação de saldo e extrato. Projeto de desenvolvimento backend, utilizando javascript, node.js, express. 

<img width=90% align="center"  src="https://github.com/patriciafraga/BancoDigital/blob/master/Anima%C3%A7%C3%A3o.gif" />

###  Desenvolvimento
O desenvolvimento do projeto contou com a elaboração de uma API RESTful e, para tanto, foram necessárias competências em javascript, node.js, manipulação de dados, manipulação de node modules como express, date-fns, criação de controladores, rotas, middlewares, funções assíncronas.
